== Description ==

Show the top commented posts per Week, Month, or All Time! No setup needed. Widget-ready. Awesome.

This very simple and effective WordPress Plugin can quickly enable you to share the top commented posts over a certain period of time (Weekly, Monthly, All Time). Use the simple widget via your WordPress dashboard and you're ready to go!

It's never been more simple or easy to increase pageviews to the posts that are already getting a lot of traffic and conversation! A must-have for highly-engaging community-oriented blogs!

<a href="http://tentblogger.com/top-posts/">Read more...</a>

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Appearance > Widgets
4. Drop the widget on your side and continue!
